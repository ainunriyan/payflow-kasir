## ğŸ¯ Quick Start Guide

### 1. Setup Awal
1. **Buka PayFlow** dan pilih "Mulai Trial 30 Hari" atau masukkan license key
2. **Buat Akun Admin** dengan username dan password yang kuat
3. **Setup Toko** - isi nama toko, alamat, dan logo
4. **Tambah Produk** - input produk pertama Anda

### 2. Transaksi Pertama
1. **Pilih Produk** dari katalog atau scan barcode
2. **Tambah ke Keranjang** dan atur quantity
3. **Input Customer** (opsional) untuk tracking
4. **Pilih Payment Method** dan proses pembayaran
5. **Cetak Struk** otomatis atau manual

### 3. Laporan Harian
1. **Buka Tab Laporan** di menu utama
2. **Pilih Tanggal** yang ingin dilihat
3. **Lihat Summary** penjualan, refund, dan profit
4. **Export Laporan** ke Excel jika diperlukan
